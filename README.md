\# Laboratorio Cloud Security



Implementación de autenticación SSH con múltiples cuentas de GitHub y sincronización automática de repositorios.



\## Descripción



Este repositorio forma parte del laboratorio de Cloud Security donde se implementó:



\- Autenticación segura mediante claves SSH

\- Uso de múltiples cuentas GitHub en un mismo equipo

\- Configuración de alias SSH

\- Push automático a repositorios personal y empresarial



\## Estructura del repositorio



README.md  -> Descripción del laboratorio  

index.md   -> Documentación técnica del proceso  

prueba.txt -> Archivo de prueba de sincronización  



\## Prueba del laboratorio



Se verificó que cada cambio realizado en el repositorio se sincroniza automáticamente con ambas cuentas GitHub usando:



git push



\## Resultado



Repositorio sincronizado con éxito en dos cuentas GitHub mediante SSH.
