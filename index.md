\# Cloud Security Lab – Multi GitHub SSH



Este laboratorio demuestra cómo trabajar con dos cuentas de GitHub usando SSH y realizar push automático a múltiples repositorios.



\## Objetivos del laboratorio



\- Configurar claves SSH para múltiples cuentas GitHub

\- Configurar archivo \~/.ssh/config

\- Conectar cuenta personal y cuenta empresa

\- Crear repositorio local

\- Realizar push a dos repositorios simultáneamente



\## Tecnologías usadas



\- Git

\- GitHub

\- OpenSSH

\- Windows PowerShell



\## Flujo de trabajo aplicado



1\. Generación de claves SSH personal y empresa

2\. Registro de claves en GitHub

3\. Configuración de alias SSH

4\. Creación de repositorio local

5\. Push a repositorio personal

6\. Push a repositorio empresa

7\. Configuración de push doble automático



\## Resultado final



Cada vez que se ejecuta el comando:



git push



El repositorio se sincroniza automáticamente con:

\- Cuenta personal GitHub

\- Cuenta empresa GitHub
